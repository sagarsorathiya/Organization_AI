Data Privacy

Your Data Never Leaves
Your Network

Unlike cloud AI services, Organization AI runs entirely within your corporate infrastructure. Every conversation, every document, every prompt stays behind your firewall.

🔒

No External API Calls

The AI model runs locally via Ollama. Prompts and responses never touch the internet. No OpenAI, no Azure, no third-party services.

Active Directory Authentication

Features

Enterprise-Grade, Out of the Box

40+ features for a production-ready AI platform — from AI agents and RAG knowledge bases to scheduled automation, all self-hosted.

🔒

AD / LDAP Single Sign-On

Authenticate users against your existing Active Directory or LDAP server. Supports LDAP and LDAPS. Role-based access via AD security groups.

🤖

Multi-Model AI Engine

Download, update, and delete AI models directly from the admin panel. Supports Llama 3, Gemma, Mistral, DeepSeek, and 27 cataloged models. Regenerate any AI response with a single click.

⚡

Real-Time Streaming

Token-by-token streaming responses with 80ms throttled rendering. Feels as fast as ChatGPT, running entirely on your hardware.

📄

Multi-File Attachments

Upload multiple files (14+ formats) in a single message: PDF, DOCX, XLSX, CSV, images, code files, and more. Admin-controllable limits per policy.

🛠

Admin Control Panel

Full web-based admin panel with 13 tabs across 3 groups: users, models, settings, audit logs, health, announcements, templates, feedback, agents, knowledge base, skills, scheduled tasks, and notifications.

📈

200 Parallel Users

Engineered for 200 simultaneous users. 16 backend workers, 100-connection DB pool, 16x parallel inference. Auto-configured by the setup wizard based on your hardware.

👍

Response Feedback

Users rate AI responses with thumbs up/down. Admins view aggregated satisfaction metrics, feedback trends, and identify areas for improvement.

📚

Prompt Templates

Admin-curated prompt template library organized by category. Users browse and apply templates directly from the chat input for consistent, high-quality prompts.

🏷

Tags, Bookmarks & Sharing

Organize conversations with custom tags and filter by tag. Bookmark important messages for quick reference. Pin, archive, and generate read-only shareable links.

📣

Announcements & MOTD

Admins create banner announcements displayed to all users. Toggle active/inactive, schedule maintenance notices, or share updates across the organization.

📊

Usage Dashboard & Export

Personal usage stats: total conversations, messages, monthly activity, top model, and uploads. Bulk export all conversations as a ZIP archive.

💡

Onboarding & Shortcuts

Interactive first-time welcome tour highlights key features. Keyboard shortcuts panel for power users. Data retention enforcement with automatic cleanup policies.

🤖

AI Agents

Create autonomous AI agents with custom system prompts, dedicated models, and configurable parameters. Agents can be shared org-wide or kept private. Full CRUD management from admin panel.

📚

RAG Knowledge Base

Upload documents (PDF, DOCX, TXT, Markdown) to knowledge bases with automatic chunking and vector embeddings. AI responses grounded in your organization's documents with semantic search.

🔧

Skills Engine

Define reusable AI skills with structured input schemas and prompt templates. Execute skills on demand with parameter validation. Track execution history and results for audit compliance.

🕐

Task Scheduler

Schedule recurring AI tasks with cron expressions — automated reports, document summarization, data extraction. APScheduler-powered with execution history and admin oversight.

📧

Notifications

In-app notification system for task completions, agent responses, and system events. Real-time delivery with read/unread tracking and bulk management actions.

📱

PWA & Memory

Install as a Progressive Web App on any device. AI memory system persists context across conversations — agents remember user preferences and organizational knowledge over time.

Quick Start

Up and Running in Minutes

One-click setup scripts detect your hardware, install dependencies, and configure everything automatically.

Windows (PowerShell) 
# Clone the repository
git clone https://github.com/sagarsorathiya/Organization_AI.git
cd organization-ai

# Run the setup wizard (detects hardware, installs everything)
powershell -ExecutionPolicy Bypass -File setup.ps1

Linux / macOS (Bash) 
# Clone the repository
git clone https://github.com/sagarsorathiya/Organization_AI.git
cd organization-ai

# Run the setup wizard
chmod +x setup.sh
./setup.sh
Docker Compose 
# Clone and configure
git clone https://github.com/sagarsorathiya/Organization_AI.git
cd organization-ai
cp .env.example backend/.env
# Edit backend/.env with your settings

# CPU-only (works on any server)
docker compose up -d

# With NVIDIA GPU acceleration
docker compose -f docker-compose.yml -f docker-compose.gpu.yml up -d
Manual Setup 
# 1. Backend
cd backend
python -m venv venv
source venv/bin/activate      # Linux
venv\Scripts\activate        # Windows
pip install -r requirements.txt
cp ../.env.example .env       # Configure .env
alembic upgrade head
uvicorn app.main:app --port 8000

# 2. Frontend
cd ../frontend
npm install
npm run dev

# 3. Pull an AI model
ollama pull llama3.1:8b
